# Educational Organisation Using ServiceNow

An Educational Management System built on the ServiceNow platform to automate student admission processing and academic progress tracking.  
The application replaces manual record handling with structured digital workflows using tables, forms, process flows, and client scripts.

---

## Overview
The system provides a centralized platform for managing student information, monitoring admission stages, and automatically calculating academic performance.  
It improves accuracy and reduces paperwork through automation.

---

## Modules

### Student Table
Stores primary student details and auto-generated admin number.

### Admission Table
Extends the student table and manages the admission lifecycle including status tracking.

### Student Progress
Records subject marks and automatically calculates total, percentage, and result.

---

## Automation Features
- Auto generation of admin numbers
- Fetch student details from admission record
- Automatic total marks calculation
- Automatic percentage and result update
- Pincode-based location autofill
- Read-only calculated fields

---

## Admission Workflow
New → In Progress → Joined → Rejoined → Rejected → Closed → Cancelled
